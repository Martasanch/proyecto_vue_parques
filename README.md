# proyecto_vue_parques
## Descripción

* Vue 3
* Consulta a la API del Ayuntamiento de Madrid
* Creación y edicción de contenido
* Integración de Mapa vue-maps con localización de coordenadas 



## Project setup

### Frontend:

* cd frontend
* npm install
* npm run serve

### Backend:

* cd backend
* npm install
* npm start


## Stack

* Sass
* JavaScript
* NodeJS
* MongoDB Atlas
* VueJS Composition API
